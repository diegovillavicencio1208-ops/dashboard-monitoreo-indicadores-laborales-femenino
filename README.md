# Dashboard - Monitoreo de Indicadores Laborales Femenino | Ecuador 2024

## Descripción
Dashboard interactivo desarrollado en Power BI a partir de datos oficiales de la ENEMDU, diseñado para facilitar el análisis del mercado laboral femenino mediante indicadores, segmentación geográfica y filtros dinámicos.

---

## Objetivo
Transformar información estadística pública en una herramienta interactiva que facilite el seguimiento de indicadores laborales y apoye el análisis de distintos grupos poblacionales.

## Características
- Indicadores laborales (KPIs)
- Segmentación por provincia, región y Ecuador
- Filtros dinámicos
- Comparación por variables sociodemográficas
- Navegación mediante botones
- Matriz de análisis interactiva

---

## Tecnologías
- Power BI
- DAX
- Power Query
- Excel

---

## Dataset
**Fuente:** Encuesta Nacional de Empleo, Desempleo y Subempleo (ENEMDU) – INEC
